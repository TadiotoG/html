ID e CLASS

ANOTAÇÕES

Utilizar # para ID
Utilizar . para class

Para ser ainda mais específico utilizar por exemplo:

h1#anota {
color: white;
}

Apenas a cor do h1 com id = anota vai ficar branca

Só pode utilizar o mesmo ID uma vez

"span" é utilizado para criar uma classe ou id em um elemento

Voce pode utilizar mais de uma classe para um elemento

O ID se sobrepõe ao CLASS pois é "ligado" depois...
